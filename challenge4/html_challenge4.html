There are 3 Types of list:

definition lists
which show terms and definitions
unordered list
which show bullet
ordered list
which show numbers or letters

Above is an example of a definition list.

Unordered list can have a type attribute which can be applied to the URL or the LI tag.
the three valuesallowed are:

  • type="square"
  • type="disc"
  • type="circle"

Ordered list can have a type attribute too.
Used along with a start attribute they can show:

  1. lower case letters
  2. upper case letters
  3. lower case roman numerals
  4. upper case roman numerals
  5. numbres
